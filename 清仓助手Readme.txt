
# 清仓助手使用方法：
 - 用调试模式运行chrome：
 chrome.exe --remote-debugging-port=9222 --user-data-dir=remote-profile

注意要在固定的目录下执行该命令。remote-profile会创建在当前目录下，这样收藏夹这些可以重复使用。
